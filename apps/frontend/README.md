# [ 2번 터미널: 프론트엔드 (웹 서버) ]

### 1. (새 터미널을 엽니다.)

### 2. frontend 폴더로 이동합니다.
```
Bash
cd frontend
```

### 3. Python의 내장 웹 서버를 5500번 포트로 실행합니다.
```
Bash
python -m http.server 5500
```
Serving HTTP on 0.0.0.0 port 5500 메시지가 뜨면 성공입니다.

[ 브라우저 확인 ]
웹 브라우저 주소창에 http://127.0.0.1:5500 을 입력하여 "AI 푸드 렌즈" 데모에 접속합니다.
